# 漏斗 v3 中立裁定方法学(Opus 逐份裁,2026-06-18)

对 `eval/out_funnel/<key>.json` 逐份独立裁定漏斗 v3 输出质量。裁判=Opus,**只信政策原文**,
auto-label(meta.label)仅作分层提示、**非真值**。三件事:R0 门、精度、召回。

## 输入(每份 json)
- `meta`:政策身份 + stratum/label(弱提示)
- `text`:政策原文(截 25k)
- `res.intervention`:引擎 R0 判定(本政策是否引入/实质改变某健康决定因素所依赖的条件)
- `res.pathways`:保留的危害,每条含 `chain`(行动→决定因素→不利结果)/`outcome_q`(1–10 初筛题)/
  `mitigation`(已含/不足/未提及)/`strength`/`measures`/`evidence`(原文锚)

## 判据

### 1. R0 干预门
HIA 对象 = **含干预的 proposed policy/program/project**:引入或实质改变某健康决定因素所依赖的条件
(建成环境/土地利用/排放/物质暴露/服务供给/资源配置/保护标准/可及性/行为塑造环境——
**含撤销服务、削减投入、放松防护标准等"非建设型"实质改变**)。
纯规则/价格/市场准入/信息披露/机构协调/资质认定 = **非干预**。
- 裁判独立判:`r0_truth`(true/false)
- 对照 `res.intervention`:`correct` / **`R0漏报`**(引擎 false 但实为干预,**重点盯·漏报最伤**)/
  `R0假阳`(引擎 true 但无干预)

### 2. 精度(有 pathways 才判)—— 逐条三判据
- **A 原文锚定**:evidence/anchor 在 text 中确有出处
- **B 干预归因**:危害源于**本政策引入的干预**,非政策仅监管/引用/沿用的**既有背景**事物
- **C 健康机制**:chain 经由中间性健康决定因素(物质环境/行为/社会心理/卫生服务)抵达**不利健康结果**,
  非停在价格/成本/效率/制度层面
三条全满足→valid;否则→invalid,注明缺陷类型:
  `硬造危害`(凭空/反转效益造) / `跨段冒充`(借别的暴露段) / `背景非干预`(既有事物非本政策引入) /
  `应激泛化`(任何摩擦→应激→心理健康的万能套) / `锚定失败`(原文查无)

### 3. 召回(r0_truth=true 才判)
通读政策,列引擎**漏掉**的显而易见不利健康影响 + 严重度(高/中/低)。无则空。

## 返回(只输出 JSON)
```json
{"key":"…","r0_truth":true,"r0_verdict":"correct|R0漏报|R0假阳",
 "n_kept":3,"n_valid":2,
 "invalid":[{"chain":"…","reason":"硬造危害"}],
 "missed":[{"harm":"…","severity":"高"}],
 "note":"1–2句总评"}
```

## 聚合指标
- **R0 准确率** = correct / 总;分列漏报、假阳清单(漏报是危害转向的命门)
- **精度** = Σn_valid / Σn_kept(全样本);按层(A 正样本 vs B 假阳 vs 负样本)分列
- **召回**:有漏报危害的政策占比 + 高severity漏报清单
- **假阳类型分布**:硬造/跨段/背景/应激/锚定 计数(指明下一步该补哪道闸)
