## 一、卡片格式(hia_evidence.py 的 CARDS 里一个 dict)

```python
{
    "keys": ["决定因素词1", "决定因素词2", ...],   # 只放"决定因素"(暴露/环境/行为/社会)词;
                                                  # 别放结果词(慢性病/中毒/伤害…已在 _STOP_KEYS 屏蔽)
    "q": "Q3",                                    # 落到的初筛题号 Q1–Q10
    "note": "一句话概括机制要点(非原文逐字,标'以链接为准')",
    "sources": [
        "文件/标准名 + 编号(年份). 发布机构. https://稳定URL",
    ],
    "status": "done",                             # done=已核实现行;todo=待补强(官网暂无/待核实)
},
```

**配套**:把这条机制里 AI 可能用的不同措辞,加进 `_SYN_GROUPS` 对应组(或新建一组),
这样换个说法也能命中(召回的主要杠杆)。例:把"电池起火/烧伤"等并入火灾组。

**分级**:`source_tier()` 已能识别 `GB/国家标准`→国家标准、`国务院/中共中央/规划纲要`→国家政策、
`技术导则/技术指南/疾控`→国内指南、WHO 各级。若新来源类型没归对,告诉我加规则。

**填空给我的最小信息**:`题号 | 决定因素(keys 候选) | 来源citation+URL | 现行性已核? | AI常用措辞(进同义词组)`

## 三、核实来源的去处(便于你查现行性)

- 中国国家标准:全国标准信息公共服务平台 `std.samr.gov.cn`、国家标准全文公开 `openstd.samr.gov.cn`
- 生态环境标准:生态环境部 `mee.gov.cn`(标准/法规栏目)
- 卫生 / 职业卫生:国家卫生健康委 `nhc.gov.cn`;法律法规:国家法律法规数据库 `flk.npc.gov.cn`
- WHO:`who.int`(fact sheets / publications);中国办公室 `who.int/china`
- 系统综述/负担:Cochrane、GBD(IHME)、中国疾控慢病监测
